Some of my projects from Vanderbilt Boot Camp

Sip and Read

This was our first group project in which we integrated HTML, CSS, JavaScript and the use of API's to create a custom website in which you can search for books and get beer recommendations. I created the logic for the Google Books API using fetch and then requests and template literals.

The Story Keepers

This was our second group project in which we integrated HTML, CSS, Javascript, Bootstrap, Node.js, Express,Bcrypt, Dotenv, MySQL2, Sequelize, Google Leapis, Handlebars, Mongo DB, Heroku. This was an evolution of the original project with a much cleaner interface and the ablitiy to Login, Create Posts about your favorite books and, View Posts, Delete Posts and Rate Posts. I did entire back end logic on this project and some of the front end.

Penny for your thoughts

This is a fake E-Commerce site that fully utilized front and back end development to purchase quotes and read the quotes that populate from Axios and Rapid api. It was created with React, React-dom, Mongoose, Node.js. Bcyrpt, Express, Graphql, Apollo, Mongo DB, Heroku, Rapid api, Axios, and Bootstrap, jsonwebtoken, Stripe, jwt-decode, faker, Express, Express-session.
